# Machine_Learning--experiment
此仓库用于上传专业选修课程——机器学习与数据挖掘实验的代码、数据以及说明文件等。

每一个实验的所有文件放进一个新建的文件夹，文件夹命名为machine_learning实验i（i为一/二/三/四/五） 
每个文件夹下面包括本实验的代码、数据、运行结果截图、实验报告 
实验报告内容包括: 1、实验方式：单干 
                 2、实验内容 
                 3、作业环境：文件说明，函数说明，调用的函数库以及涉及哪些技术 
                 4、难题与解决 
                 5、实验总结
